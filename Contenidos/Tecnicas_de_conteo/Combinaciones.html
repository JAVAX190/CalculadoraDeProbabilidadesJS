Combinaciones

Selección de elementos sin importar el orden

Las combinaciones se aplican cuando se seleccionan algunos elementos de un conjunto, pero el orden no importa.

Variables: n = total de elementos, r = elementos seleccionados.
Fórmula: C(n, r) = n! / [r!(n - r)!]
