Permutaciones con r

Selección ordenada de subconjuntos

Las permutaciones con r se usan para determinar cuántas formas diferentes existen para organizar r elementos dentro de un conjunto total de n elementos.

Variables: n = total de elementos, r = elementos seleccionados.
Fórmula: P(n, r) = n! / (n - r)!
